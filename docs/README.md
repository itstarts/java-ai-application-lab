# 文档入口

本目录保存仓库内的工程文档。仓库中的文档是工程执行和 AI 接手的权威来源；个人知识库或飞书中的副本只用于阅读和导入。

## 阅读顺序

涉及阶段推进、计划调整、跨会话接手或较大功能变更时，按以下顺序阅读：

1. [`../README.md`](../README.md)
2. [`../AGENTS.md`](../AGENTS.md)
3. [`iterations/STATUS.md`](iterations/STATUS.md)
4. [`reference/engineering-baseline.md`](reference/engineering-baseline.md)
5. [`roadmap/java-ai-learning-roadmap.md`](roadmap/java-ai-learning-roadmap.md)
6. [`iterations/chat-api.md`](iterations/chat-api.md)
7. [`iterations/backlog.md`](iterations/backlog.md)
8. [`iterations/decisions.md`](iterations/decisions.md)

当前阶段是阶段 1，执行文档是 `docs/iterations/chat-api.md`。阶段 0-8 的补充文档用于学习和阶段设计；当前阶段必须实现的硬约束以工程基线和当前迭代文档为准。

## 文档分工

| 目录 | 用途 |
|---|---|
| `roadmap/` | 总学习路线和项目演进路线 |
| `reference/` | 当前已经生效的跨阶段工程契约 |
| `stages/` | 后续阶段指南和设计参考 |
| `iterations/` | 当前状态、阶段推进计划、待办和决策记录 |
| `notes/` | 阶段学习笔记 |

阶段 taxonomy 以 `roadmap/java-ai-learning-roadmap.md` 为准。README 只保留入口，不维护第二份完整阶段表。
